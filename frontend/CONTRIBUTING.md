# Contributing

This is a **demo / portfolio project** — an archive of a Learning Management
System built for Halaqoh Cinta Qur'an. It is no longer actively used and is
shared publicly as a portfolio demonstration.

While contributions are not actively sought, you are welcome to:

- **Report bugs** via GitHub Issues
- **Suggest improvements** via GitHub Issues
- **Fork** the project for your own learning or adaptation

Please note that pull requests may not be reviewed or merged in a timely
manner, as this project is maintained on a best-effort basis.
